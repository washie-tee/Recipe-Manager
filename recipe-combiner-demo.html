🛒 Recipe Combiner Module

The Recipe Combiner module allows users to select multiple recipes and automatically generate a consolidated shopping list with combined ingredients. Perfect for meal planning and bulk cooking!

✨ Key Features

  • Multi-Recipe Selection: Choose multiple recipes from your collection
  • Recipe Scaling: Adjust serving sizes with multipliers (e.g., 2x, 0.5x)
  • Smart Ingredient Consolidation: Automatically combines similar ingredients
  • Unit Normalization: Handles different units (cups, tbsp, tsp, etc.)
  • Categorized Shopping List: Organizes ingredients by category (Produce, Dairy, etc.)
  • Print & Export: Generate printable lists or export as text files
  • Quantity Parsing: Handles fractions, decimals, and mixed numbers
  • Real-time Updates: Shopping list updates as you modify selections

🔧 How It Works

The module uses advanced ingredient parsing to understand quantities, units, and ingredient names. It then consolidates similar ingredients across recipes and generates organized shopping lists.

Example Ingredient Parsing:

Input: "2 1/2 cups all-purpose flour" Parsed: { quantity: 2.5, unit: "cup", ingredient: "all-purpose flour" } Input: "1 tbsp olive oil" Parsed: { quantity: 1, unit: "tbsp", ingredient: "olive oil" }

🎯 Usage Instructions

  1. Click the "🛒 Combine Recipes" button in the main interface
  2. Search and select recipes from the left panel
  3. Adjust serving multipliers as needed
  4. Click "📝 Generate List" to create your shopping list
  5. Use "🖨️ Print" or "💾 Export" to save your list
